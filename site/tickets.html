Gut zu wissen

Abendkasse & Einlass

Die Abendkasse öffnet eine Stunde vor Vorstellungsbeginn. Wir haben keine Barkasse, aber Sie können mit EC- und Kreditkarte sowie kontaktlos zahlen.

Bereits gekaufte Karten können wir leider nicht umtauschen oder zurückgeben. Nach Vorstellungsbeginn ist kein Einlass möglich.

Barrierefreier Platz für Rollstuhlfahrer

Es befindet sich ein barrierefreier Rollstuhlplatz in der ersten Reihe – im Saalplan des Ticketshops mit dem Rollstuhlsymbol gekennzeichnet. Die Begleitperson bitte in der Nähe einbuchen.

So lösen Sie einen Gutschein-Code ein
  1. Klicken Sie im Ticketshop auf das gewünschte Datum der Vorstellung.
  2. Legen Sie die gewünschte Anzahl Tickets in den Warenkorb.
  3. Klicken Sie im Warenkorb auf den Button „Ich habe einen Gutscheincode“ unterhalb der Gesamtsumme.
  4. Geben Sie den oder die Codes nacheinander ein und klicken Sie auf „Einlösen“.

Gutscheine enthalten einen Code und können nur im Online-Ticketshop eingelöst werden, nicht an der Abendkasse.

So nutzen Sie ein Passwort (z. B. Mitarbeiterrabatt)
  1. Klicken Sie im Ticketshop auf das gewünschte Datum der Vorstellung.
  2. Klicken Sie unterhalb der Kategorieauswahl auf „Ich habe ein Passwort“.
  3. Geben Sie Ihr Passwort ein und klicken Sie auf „Absenden“. Die rabattierte Kategorie wird Ihnen unter den Preiskategorien angezeigt.
  4. Legen Sie die gewünschte Anzahl Tickets in den Warenkorb und wählen Sie dort die Rabattkategorie aus.
