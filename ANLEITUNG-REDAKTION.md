# austroPott-Website selbst bearbeiten – Anleitung

Diese Anleitung richtet sich an die Redaktion (z. B. Michael) und erklärt,
wie man Inhalte der Website ohne technische Kenntnisse ändert.

## So kommen Sie in den Redaktionsbereich

1. Im Browser öffnen: **https://christiansrimpler-create.github.io/austropott-website/admin/**
   (nach dem Umzug auf die eigene Domain: https://www.austropott.de/admin/)
2. Auf **„Sign In Using Access Token“** klicken und den Zugangsschlüssel einfügen
   (wird bei der Einrichtung einmal gemeinsam hinterlegt – der Browser merkt ihn sich danach).

## Was Sie bearbeiten können

| Bereich | Was darin steckt |
|---|---|
| **Termine** | Vorstellungstermine: Datum, Uhrzeit, Stück. Vergangene Termine verschwinden automatisch. |
| **Stücke** | Die Stück-Seiten: Texte, Fotos, Zitate. Neue Stücke können angelegt, alte gelöscht werden. |
| **Startseite & Zitate** | Überschrift und Text auf dem großen Bild, Pressezitat, Publikumsstimmen, Kontaktdaten, Sponsoren-Logos. |
| **Team** | Die Team-Seite: Namen, Fotos, Lebensläufe. |
| **Ticketpreise** | Die Preistabelle und die Ermäßigungen. |
| **Galerie** | Die Fotos in der Galerie (neue hochladen, alte entfernen, Reihenfolge ändern). |
| **Archiv** | Die Liste vergangener Produktionen. |

## Einen neuen Termin eintragen (Beispiel)

1. Links auf **Termine** klicken, dann den Eintrag **Vorstellungstermine** öffnen.
2. Auf **„Add“ / Plus** bei „Termine“ klicken.
3. Datum auswählen, Uhrzeit eintragen (z. B. `19:30`), Stück aus der Liste wählen.
4. Oben rechts auf **„Save“ (Speichern)** klicken.

Nach dem Speichern baut sich die Website automatisch neu –
**nach 1–2 Minuten** ist die Änderung online. Einfach die Seite im Browser
neu laden (ggf. mit Strg+F5 / Cmd+Shift+R).

## Wichtig zu wissen

- **Nichts kann dauerhaft kaputtgehen.** Jede Änderung wird protokolliert und
  lässt sich wiederherstellen. Im Zweifel: anrufen, wir setzen es zurück.
- Fotos am besten als JPG, nicht größer als ein paar MB. Beim Hochladen im
  Feld „Foto“ auf „Upload“ klicken.
- Zitate bitte **ohne** Anführungszeichen eintragen – die setzt die Website
  automatisch.
